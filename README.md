# Aloware SMS to CRM Timeline Sync

**Description: Bridges the gap between the VoIP provider (Aloware) and the CRM. Ensures support agents have full visibility of SMS communication history within the deal view.**

Key Features:

Regex Parsing: Extracts the agent's name from the Aloware message body to attribute the message correctly in the CRM.

Fallback Logic: Implements logic to handle "Incoming" vs "Outgoing" message directions.
